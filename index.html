Komitmen Privasi

Saya, Muhaajir, berkomitmen melindungi privasi Anda dengan transparan penuh. Data Anda tidak pernah dijual, disewakan, atau dibagikan untuk keperluan pemasaran kepada pihak ketiga mana pun. Setiap interaksi di situs ini dirancang dengan prinsip privacy-first.

1. Data yang Dikumpulkan

Kami mengumpulkan data berikut secara minimal:

  • Informasi Sukarela — nama dan pesan yang Anda kirimkan melalui formulir kontak.
  • Data Teknis Anonim — alamat IP, jenis browser, dan sistem operasi melalui Google Analytics (dalam bentuk agregat dan anonim).
  • Preferensi Lokal — pilihan bahasa dan tema disimpan di browser Anda melalui localStorage, bukan di server.
  • Tools Digital — semua aplikasi (HajirSync, Muhaajir Delta, Scrub, Vault, RAIA) berjalan sepenuhnya di sisi klien dan tidak mengirim data pribadi ke server mana pun.

2. Cookies & Pihak Ketiga

Kami menggunakan:

  • Cookies Fungsional — untuk menyimpan preferensi Anda (bahasa, tema) agar pengalaman lebih nyaman.
  • Google Analytics — dengan anonimisasi IP aktif, hanya untuk memahami pola penggunaan agregat.
  • Formulir Kontak — diproses oleh Formspree; data dihapus maksimal 30 hari setelah pengiriman.
  • Tautan Eksternal — seperti YouTube, Instagram, SoundCloud, dan GitHub, tunduk pada kebijakan privasi masing-masing platform.

3. Keamanan Data

  • Seluruh situs menggunakan HTTPS/TLS (enkripsi end-to-end).
  • Content Security Policy (CSP) aktif untuk mencegah serangan XSS.
  • Validasi input dan pembatasan akses data secara ketat.
  • Semua tools berjalan di browser Anda — data tidak pernah meninggalkan perangkat Anda.

4. Hak Anda

Anda berhak untuk:

  • Mengakses — meminta salinan data pribadi yang kami simpan.
  • Memperbaiki — mengoreksi data yang tidak akurat.
  • Menghapus — meminta penghapusan data pribadi Anda.

Untuk menggunakan hak-hak tersebut, silakan hubungi melalui formulir kontak di halaman utama.

Ringkasan: Kami mengumpulkan data seminimal mungkin, tidak pernah menjual data Anda, dan semua tools berjalan lokal di perangkat Anda. Privasi Anda adalah prioritas kami.

Kebijakan Cookie

Versi 3.8.a — 23 Juni 2026

Kami menggunakan cookie dan teknologi serupa untuk meningkatkan pengalaman Anda di situs ini. Kebijakan ini menjelaskan jenis cookie yang kami gunakan, tujuan penggunaannya, serta pilihan yang Anda miliki.

1. Apa itu Cookie?

Cookie adalah file teks kecil yang disimpan di perangkat Anda (komputer, ponsel, tablet) oleh situs web yang Anda kunjungi. Cookie membantu situs mengingat preferensi Anda dan memberikan pengalaman yang lebih personal.

2. Cookie yang Kami Gunakan

Nama Cookie Tipe Tujuan Masa Berlaku
raia_state_v1 Fungsional Menyimpan konfigurasi RAIA (provider, model, parameter) Permanen (hingga dihapus)
mhj_lang Preferensi Menyimpan pilihan bahasa (ID/EN) di berbagai tools Permanen (hingga dihapus)
theme Preferensi Menyimpan preferensi tema (gelap/terang) Permanen (hingga dihapus)
hajir-cookie Fungsional Menyimpan persetujuan cookie banner Permanen (hingga dihapus)
_ga (Google Analytics) Analitik Mengidentifikasi pengunjung unik (anonim) 2 tahun
_ga_* (Google Analytics) Analitik Menyimpan status sesi dan data agregat 2 tahun

Cookie fungsional dan preferensi hanya digunakan untuk meningkatkan pengalaman Anda. Data analitik dianonimkan dan tidak digunakan untuk melacak Anda secara individual.

3. Cookie Pihak Ketiga

Kami menggunakan layanan pihak ketiga terpercaya yang juga menempatkan cookie:

  • Google Analytics — untuk analisis lalu lintas agregat. Data dianonimkan dan tidak dibagikan dengan pihak lain.
  • Formspree — untuk pemrosesan formulir kontak (tidak menempatkan cookie pelacakan).
  • YouTube, Instagram, SoundCloud, GitHub — tautan eksternal yang tunduk pada kebijakan cookie masing-masing platform.

4. Kontrol Cookie

Anda dapat mengontrol dan mengelola cookie dengan beberapa cara:

  • Melalui browser: Sebagian besar browser memungkinkan Anda menolak atau menghapus cookie melalui pengaturan privasi.
  • Melalui banner cookie: Saat pertama kali mengunjungi situs, Anda dapat memilih untuk menyetujui atau menolak cookie non-fungsional.
  • Google Analytics Opt-out: Anda dapat menginstal add-on opt-out untuk menonaktifkan pelacakan Google Analytics.

5. Perubahan Kebijakan Cookie

Kami dapat memperbarui kebijakan ini dari waktu ke waktu untuk mencerminkan perubahan dalam praktik kami. Versi terbaru akan selalu tersedia di halaman ini dengan tanggal pembaruan yang tertera.

Ringkasan: Kami menggunakan cookie fungsional untuk kenyamanan Anda, cookie preferensi untuk pengaturan bahasa & tema, serta analitik anonim untuk meningkatkan situs. Anda selalu dapat mengontrol cookie melalui pengaturan browser atau banner persetujuan.
