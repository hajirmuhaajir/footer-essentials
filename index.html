Komitmen Privasi

Saya, Muhaajir, berkomitmen melindungi privasi Anda dengan transparan. Data Anda tidak pernah dijual, disewakan, atau dibagikan untuk keperluan pemasaran.

1. Data yang Dikumpulkan

Informasi sukarela (nama, pesan) melalui formulir kontak. Data teknis anonim (IP, browser) via Google Analytics. Preferensi lokal (bahasa, tema) disimpan di browser Anda. Tools digital tidak mengirim data pribadi ke server.

2. Cookies & Pihak Ketiga

Cookies fungsional untuk menyimpan preferensi. Google Analytics (anonim). Formulir diproses oleh Formspree (data dihapus maksimal 30 hari). Link eksternal tunduk pada kebijakan masing-masing.

3. Keamanan Data

Seluruh situs menggunakan HTTPS/TLS. Content Security Policy (CSP) aktif. Validasi input dan akses data sangat dibatasi.

4. Hak Anda

Anda berhak mengakses, memperbaiki, atau menghapus data pribadi Anda. Untuk menggunakan hak tersebut, hubungi melalui formulir kontak di halaman utama.
